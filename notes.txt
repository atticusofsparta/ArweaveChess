What is Arweave Chess?
    This is a multiplayer chess game using socket IO and atomic nfts on arweave. 
    Players are required to mint a membership nft and can then mint chess sets.
    Wallet is Arconnect.

Website is hosted via Ens + arweave  with the server hosted via heroku.
